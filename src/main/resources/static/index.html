🤖 AI角色扮演系统

基于Spring Boot和阿里云语音服务的智能对话系统
支持多种角色扮演、语音转录和实时对话功能

🎭

角色扮演

支持多种预设角色，包括萌妹子、专业顾问等，每个角色都有独特的性格和说话风格

🎤

语音转录

集成阿里云ASR服务，支持实时语音转文字，让对话更加自然流畅

💬

智能对话

基于通义千问大模型，提供智能、有趣的对话体验，支持流式输出

🔊

语音合成

集成阿里云TTS服务，支持将AI回复转换为自然语音，实现真正的语音对话

🎯

完整语音聊天

完整的语音交互体验：录音→转录→AI回复→语音合成→播放，一站式语音对话解决方案

🎯 完整语音聊天 🎤 语音转录测试 💬 简单对话 🎭 角色扮演

📡 API接口说明

GET /api/chat/simple/chat - 简单AI对话
GET /api/chat/roleplay/simple/chat - 角色扮演对话
GET /api/chat/stream/chat - 流式对话
POST /api/speech/transcribe - 语音转录
POST /api/voice-chat/process - 完整语音聊天（录音→转录→AI回复→语音合成）
POST /api/voice-chat/tts - 文本转语音合成
POST /api/voice-chat/transcribe - 语音转录
POST /api/voice-chat/chat - 文本对话
GET /api/speech/health - 语音服务健康检查
